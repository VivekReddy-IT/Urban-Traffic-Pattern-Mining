# 🚦 Urban Traffic Pattern Mining using Data Mining Techniques

## 📌 Overview

This project analyzes urban traffic data to discover hidden patterns using Data Mining techniques such as Association Rule Mining (Apriori) and Clustering (K-Means). It also includes an interactive web application built using Streamlit.

## 🎯 Objectives

* Discover traffic patterns based on time and weather
* Apply Apriori algorithm for association rule mining
* Use K-Means clustering to group traffic conditions
* Build an interactive UI for analysis

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* mlxtend
* Streamlit

## 📊 Key Features

* Traffic pattern analysis
* Association rule mining
* Clustering of traffic conditions
* Interactive Streamlit web app

## 📈 Results

* Early morning → Low traffic (high confidence)
* Mid-day → High traffic congestion
* Clear clustering into low, medium, high traffic

## 📂 Dataset

Metro Interstate Traffic Volume Dataset (Kaggle)

## 🚀 How to Run

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the Streamlit app:

```
streamlit run app.py
```
